# Project 02 - Cube 

In this project you can draw a cube with rotations. This project has users interactions.

### Speed kyes: 
   - '+' : plus 0.1
   - '-' : minus 0.1 
  

### Zoom and Positions 
  -  Up arrow : Zoom out
  - Down arrow : Zoom in
  - Left arrow : Translate to left
  - Right arrow :  Translate to right
